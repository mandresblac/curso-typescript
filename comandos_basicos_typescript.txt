Nota: Para instalar Typescript en mi equipo priemro debo tener instalado Node.js

npm install -g typescript  ---> Instala typescript globalmente
npm install typescript --save-dev  --->  Instala typescript en un proyecto especifico

tsc  --->  Comando de Typescript que compila todos los archivos .ts en la carpeta actual y genera un archivo .js para cada uno, tambien sirve para comprobar que Typescript esta instalado correctamente en su equipo.

tsc -v  --->  Para saber la version de typescript que tenemos instalada

tsc --init  --->  Inicializa un proyecto TypeScript y crea un archivo tsconfig.json.
tsc -w --->  La bandera -w de (watch) es para que observe los cambios, no de uno sino de todos los archivos de un proyecto, automaticamente.
